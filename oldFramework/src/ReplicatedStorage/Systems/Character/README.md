This folder contains the complete character system. You can move this entire folder independently.

## What's Inside

- **CharacterLocations.lua** - Helper functions to find character parts
- **CharacterUtils.lua** - Character utility functions
- **CrouchUtils.lua** - Crouch mechanics and utilities
- **RagdollSystem.lua** - Ragdoll physics system
- **RigManager.lua** - Rig management and tracking
- **RigRotationUtils.lua** - Rig rotation utilities

## Usage

Character creation is handled by `CharacterService` on the server and `CharacterSetup` on the client.

## Dependencies

- Uses `Config.Gameplay.Character` for settings
- Uses `Locations` for module paths
